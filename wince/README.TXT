The Windows CE build environment is a Solution file for Microsoft 
Visual C++ 2005 (wince/hamsterdb/hamsterdb.sln). It compiles a static and 
a dynamic library, and a sample (db1).

The static library is built in '/wince/hamsterdb/release/libhamsterdb.lib', 
the dynamic library is in '/wince/hamsterdb/release/hamsterdb.dll'. An import
library is also provided. 

The project files are configured to work with the Windows CE x86 reference 
platform.
